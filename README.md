# saulimedes-overlay

Personal Gentoo overlay.

## Usage

```bash
eselect repository add saulimedes-overlay git https://github.com/saulimedes/saulimedes-overlay.git
emaint sync -r saulimedes-overlay
```

## Packages

| Package | Description |
|---------|-------------|
| dev-vcs/got | [Game of Trees](https://gameoftrees.org/) - OpenBSD version control system (portable) |
